# Personal Info Manager

## 📌 Description

This is a beginner Python project that collects user information and displays it in a structured format.

## 🚀 Features

* Takes user input (Name, Age, Height, Qualification, Hobby, Interests)
* Uses different data types (int, float, string)
* Calculates estimated birth year
* Displays formatted output
  

## 🛠️ Tech Used

* Python

## ▶️ How to Run

1. Open the file in VS Code
2. Run the program
3. Enter your details when prompted

## 📷 Sample Output

Enter your Name: Diya
Enter your Age: 20
Enter your Height (in cms): 165

----- User Information ------

Name: Diya
Age: 20
Height: 165 cm
Estimated Birth Year: 2006
